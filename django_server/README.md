# Setup
## Requirement
- Python 3.9
- Os: Window
## Run

- Go to folder `django_server`
- Run to install package `pip install -r requirements.txt`
- Start server `python ./manage.py runserver 8000`

Server Django run at <http://127.0.0.1:8000>
